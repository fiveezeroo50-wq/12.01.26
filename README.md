Telpu nomu sistēma

Projekta apraksts

Šis projekts ir interaktīva mājaslapa telpu nomu aprēķināšanai. Sistēma ļauj lietotājiem aprēķināt telpu nomu cenu, pamatojoties uz stundu skaitu un dienas veidu (darba diena vai brīvdiena). Mājaslapa ir izstrādāta, izmantojot HTML, CSS un JavaScript, nodrošinot dinamisku un interaktīvu lietotāja pieredzi.

Funkcionalitātes skaidrojums

Pamatfunkcionalitāte
Stundu skaita ievade: Lietotājs var ievadīt vēlamo stundu skaitu telpas nomai.
Dienas veida izvēle: Lietotājs var izvēlēties starp darba dienu un brīvdienu.
Cenas aprēķins: Sistēma automātiski aprēķina kopējo cenu:
Darba dienas tarifs: 25 EUR/stundā
Brīvdienas tarifs: 35 EUR/stundā

Papildu funkcionalitāte
Datu validācija: Pārbauda, vai visi lauki ir pareizi aizpildīti.
Dinamiska stila maiņa: Pogas krāsa un teksts mainās atkarībā no izvēlētā dienas veida.
Vizuāla atgriezeniskā saite: Rezultāts tiek attēlots ar noformējumu, kas atšķiras darba dienām un brīvdienām.
Kļūdu apstrāde: Parāda skaidrus kļūdu ziņojumus, ja dati nav ievadīti pareizi.

Lietošanas instrukcijas

Atveriet mājaslapu: Atveriet failu `index.html` tīmekļa pārlūkprogrammā.
Ievadiet stundu skaitu: Ievadiet vēlamo stundu skaitu telpas nomai (minimālais skaits: 1).
Izvēlieties dienas veidu: Izvēlieties "Darba diena" vai "Brīvdiena" no izvēlnes.
Nospiediet pogu: Nospiediet pogu "Aprēķināt cenu".
Skatiet rezultātu: Rezultāts tiks parādīts zem formas ar detalizētu informāciju par cenu.

Piemēri
Darba diena, 4 stundas: 4 × 25 EUR = 100 EUR
Brīvdiena, 3 stundas: 3 × 35 EUR = 105 EUR
